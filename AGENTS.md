# AGENTS.md

## リポジトリ構成
- `apps/web`: Next.js フロントエンド実装
- `apps/api`: NestJS API 実装
- `packages/shared`: フロント・API 共有型
- `prisma`: Prisma schema

## 実装ルール
- TypeScript で型を明示し、`any` は使用しない
- 変更は最小単位で行い、将来の拡張を阻害しない設計を優先する
- 不要なコメントや過剰なボイラープレートは避ける

## 開発プロセス
- 作業は次の順で進める
  1. 変更方針
  2. 変更ファイル
  3. 実装
  4. 確認事項

## PR / コミット運用
- PR本文は日本語で作成する
- コミットメッセージは日本語で作成する
